# CyberGuard NG

**CyberGuard NG** – an AI‑powered security dashboard tailored for Nigerian SMEs. The application provides real‑time breach monitoring, scam detection, employee training, and compliance tools, all wrapped in a clean, light‑theme UI (white background with light‑teal accents).

## Features
- Dashboard with breach monitoring, scam detection, and alerts
- Interactive training module for staff awareness
- Compliance checklist for NDPA regulations
- Dark‑first design system now using a light‑theme (white background) with teal accents

## Getting Started
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit `http://localhost:5173` to explore the app.
